# 🛡️ PhishGuard AI
## 피싱·스캠 예방을 위한 다중 채널 실시간 AI 플랫폼

---

## 1. 서비스 개요

### 1.1 배경
최근 피싱·스캠 범죄는 AI 기술을 활용하여 점점 더 정교해지고 있습니다. 딥페이크를 이용한 CEO 사칭 사기(2024년 2,500만 달러 피해 사례), AI 챗봇을 활용한 실시간 피싱 공격, 초개인화된 스미싱 메시지 등 기존의 단순 키워드 기반 필터링으로는 대응이 불가능한 공격이 급증하고 있습니다.

### 1.2 문제 정의
| 문제점 | 현황 |
|--------|------|
| 정교한 피싱 메시지 | AI가 문법 오류 없는 완벽한 메시지 생성 |
| 사후 대응 중심 | 피해 발생 후 신고·차단 |
| 사용자 인식 부족 | 왜 위험한지 모르고 당함 |
| 분산된 채널 | SMS, 이메일, 메신저 각각 별도 대응 |

### 1.3 솔루션: PhishGuard AI
> **"의심하는 순간, 바로 검증하는"** 실시간 AI 기반 피싱·스캠 예방 플랫폼

**핵심 가치:**
- 🔍 **실시간 분석**: 메시지 수신 즉시 위험도 평가
- 💡 **설명 가능한 AI**: 왜 위험한지 자연어로 설명
- 📚 **능동적 교육**: 시뮬레이션 기반 사용자 학습

## 2. 핵심 기능

### 2.1 📱 다중 채널 실시간 분석 엔진

**지원 채널:**
- SMS/MMS 문자 메시지
- 카카오톡, 텔레그램 등 메신저
- 이메일 (Gmail, Outlook 연동)

**분석 요소:**

| 분석 항목 | 세부 내용 | 기술 |
|----------|----------|------|
| 텍스트 의미 분석 | 긴급성 유도, 금전 요구 패턴 | KoBERT NLP |
| URL 안전 검사 | 피싱 사이트, 단축 URL 해제 | XGBoost + DB 연동 |
| 발신자 신뢰도 | 스팸 DB, 신고 이력 조회 | 외부 API 연동 |
| 문맥 분석 | 사칭, 위장 패턴 탐지 | Transformer |

### 2.2 🤖 설명 가능한 AI (Explainable AI)

기존 피싱 탐지 서비스의 가장 큰 한계는 "왜 위험한지" 알려주지 않는다는 점입니다. PhishGuard AI는 LLM 기반 설명 생성기를 통해 사용자에게 명확한 판단 근거를 제공합니다.

**예시:**

```
📨 수신 메시지:
"[국민은행] 고객님의 계좌가 정지되었습니다. 
본인 확인을 위해 아래 링크를 클릭하세요. 
https://kookmin-bnak.com/verify"

🚨 PhishGuard AI 분석 결과:
━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ 위험도: 95/100 (매우 위험)

🔍 탐지된 위험 요소:
1. [URL 위장] 'kookmin-bnak.com'은 'kookmin-bank.com'을 
   모방한 피싱 도메인입니다 (Typosquatting)
2. [긴급성 유도] "계좌 정지" - 공포심을 유발하여 
   즉각적인 행동을 유도하는 전형적인 피싱 패턴
3. [개인정보 요구] 본인 확인을 명목으로 
   민감한 정보 입력을 유도

💡 권장 조치:
• 링크를 클릭하지 마세요
• 국민은행 공식 앱(☎ 1599-9999)으로 직접 확인하세요
• 이 메시지를 스팸으로 신고해주세요
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 2.3 🎓 AI 맞춤형 피싱 교육 시뮬레이션

**게이미피케이션 기반 학습 시스템:**

1. **진단 테스트**: 사용자의 피싱 취약점 분석
2. **맞춤형 시나리오**: 취약 유형에 맞는 모의 피싱 체험
3. **즉각적 피드백**: 잘못된 판단 시 상세 설명 제공
4. **레벨 시스템**: 학습 진도에 따른 보안 등급 부여

```
🎮 피싱 방어력 테스트
━━━━━━━━━━━━━━━━━━━━━━
현재 레벨: 🥉 초급 (Bronze)
방어력 점수: 68/100

📊 취약 유형 분석:
• 택배 사칭: ⚠️ 취약
• 금융기관 사칭: ✅ 양호  
• 정부기관 사칭: ⚠️ 취약

🎯 오늘의 미션:
"택배 사칭 피싱 5개 구별하기"
[시작하기]
━━━━━━━━━━━━━━━━━━━━━━
```

### 2.4 📊 커뮤니티 기반 위협 공유

- 사용자 신고 기반 실시간 위협 DB 업데이트
- 지역별/연령별 피싱 트렌드 대시보드
- 가족/그룹 보호 기능 (고령 가족 보호)
