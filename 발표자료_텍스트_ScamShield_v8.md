## 1. 문제 정의

### 피싱·스캠 피해 현황
- 2023년 보이스피싱 피해액: **7,744억원** (금융감독원)
- 피해자 60%가 **50대 이상 고령층**
- 가족 사칭 피싱이 전체의 **42%**

### 기존 AI 탐지의 한계
- 텍스트만으로 판단 시 **도메인 누수** 발생
- "엄마 돈 보내줘"가 실제 가족인지, 사기인지 구분 불가
- AUC 1.00 = 학습 데이터 편향 (뉴스 vs 문자)

## 3. 동일 텍스트, 다른 결과

### 예시: "엄마 급히 돈 좀 보내줘"

**Case A: 모르는 번호에서 온 경우**
- 발신자유형: +25 (unknown)
- 연락처등록: +20 (미등록)
- 첫연락: +15
- 긴급키워드: +5 ("급히")
- **메타 스코어: 65점 = 🟠 위험**

**Case B: 등록된 연락처에서 온 경우**
- 발신자유형: 0 (registered)
- 연락처등록: 0 (등록됨)
- 첫연락: 0 (기존 대화)
- 긴급키워드: +5
- **메타 스코어: 5점 = 🟢 안전**

## 4. XAI 분석 결과

### v7 모델의 문제점 발견

**피싱으로 판단한 근거:**
- "담당자", "배정", "요청금액", "합니다"
- → **격식체 = 피싱**으로 학습

**정상으로 판단한 근거:**
- "세상에", "되더라고요", "ㄷㄷ"
- → **구어체 = 정상**으로 학습

### v8 해결
- AI 점수 비중 축소 (50점 만점)
- 메타 정보 비중 확대 (110점 만점)
- 텍스트 편향을 컨텍스트로 우회
