# 🛡️ ScamShield Family v8.2
## AI 기반 가족 보호형 피싱·스캠 예방 플랫폼

---

## 1. Summary

**ScamShield Family**는 AI 모델과 메타정보 분석을 결합한 **100점 만점 실시간 피싱 탐지 시스템**입니다.

- **핵심 가치**: 텍스트 내용 + 발신자 정보 + 행동 패턴을 종합 분석하여 피싱 위험도를 정량화
- **차별점**: 등록된 연락처 여부, 첫 연락 여부 등 **메타정보 기반 스코어링**으로 정확도 향상
- **타겟 사용자**: 피싱에 취약한 고령층 및 이를 보호하려는 가족 구성원
- **점수 체계**: AI 30점 + 메타 스코어 70점 = 100점 만점

## 2. 문제 정의 (Problem Definition)

### 2.1 현황
| 지표 | 수치 | 출처 |
|------|------|------|
| 연간 보이스피싱 피해액 | **7,744억원** | 금융감독원 (2023) |
| 60대 이상 피해 비율 | **47.8%** | 경찰청 (2023) |
| 스미싱 신고 건수 | **50만건+** | KISA (2023) |
| 미등록 번호 발신 피싱 비율 | **80%** | 금융감독원 (2023) |

### 2.2 기존 솔루션의 한계
1. **텍스트만 분석**: 발신자 정보, 대화 이력 등 컨텍스트 무시
2. **이진 분류**: 피싱/정상의 단순 구분 → 경계 케이스 대응 불가
3. **사후 대응**: 피해 발생 후 신고/차단 중심
4. **개인 보호 중심**: 가족 연계 보호 시스템 부재

### 2.3 핵심 문제
> "같은 메시지라도 **누가, 어떤 상황에서** 보냈는지에 따라 위험도가 달라진다"

예시:
- ❌ 모르는 번호: "엄마 급히 돈 좀 보내줘" → **피싱 가능성 높음**
- ✅ 등록된 자녀: "엄마 급히 돈 좀 보내줘" → **정상 가능성 높음**

## 4. 주요 기능 정의 (Key Features)

### 4.1 AI 기반 피싱 탐지
- **모델**: KoELECTRA (한국어 특화 언어모델)
- **학습 데이터**: 메신저 피싱 + 보이스피싱 전사 데이터
- **출력**: 피싱 확률 (0~100%) → 30점 만점으로 환산

### 4.2 메타 스코어링 시스템 (7개 항목)

| 항목 | 점수 | 근거 |
|------|------|------|
| 모르는 번호 | +18점 | 피싱 80%가 미등록 발신 |
| 연락처 미등록 | +15점 | 미등록 시 피해 4.2배 증가 |
| URL 포함 | +12점 | 스팸 67%가 URL 포함 |
| 첫 연락 | +10점 | 첫 연락 사기 78% |
| 금융 키워드 | +10점 | 피싱 92%가 금융 키워드 포함 |
| 전화번호 포함 | +7점 | 콜백 유도 43% |
| 긴급 키워드 | +6점 | 긴박감 조성 패턴 |

### 4.3 위험 등급 분류

| 등급 | 점수 | 조치 |
|------|------|------|
| 🟢 **안전** | 0-24점 | 정상 메시지 |
| 🟡 **주의** | 25-49점 | 주의 안내 표시 |
| 🟠 **위험** | 50-74점 | 보호자 알림 전송 |
| 🔴 **긴급** | 75점+ | 즉시 차단 + 긴급 알림 |

### 4.4 Multi-Guardian 보호자 알림
- 우선순위별 보호자 등록 (예: 1순위 아들, 2순위 딸)
- 위험 등급에 따른 자동 알림 발송
- 긴급 상황 시 즉각 개입 가능
